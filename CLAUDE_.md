# Project Objective — Bank Transfer Service

The objective of this project is to simulate a simplified banking transfer system using modern backend engineering practices commonly adopted in financial institutions.

The project was designed to demonstrate technical knowledge in:

* Java 17
* Spring Boot
* Hexagonal Architecture
* Domain-Oriented Design
* Event-Driven Architecture with Kafka
* Relational persistence with PostgreSQL
* Audit persistence with MongoDB
* REST API development
* Transaction consistency
* Automated testing
* Dockerized environments

The system allows:

* account creation
* balance consultation
* money transfers between accounts
* asynchronous event publishing
* audit log persistence

The main focus is not building a complete banking platform, but demonstrating:

* clean architecture
* separation of concerns
* scalability concepts
* asynchronous communication
* maintainability
* backend engineering best practices

The scope was intentionally kept small and pragmatic to prioritize code quality, architectural clarity and business flow simplicity.
